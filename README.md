# RTv1
42 ray tracing project

## Download

Clone:

`git clone git@github.com:AlexeyBukin/RTv1.git RTv1`

Then load submodules:

`cd RTv1 && git submodule update --init --recursive`

## Build
### Mac:

Make:

`make OS=macos`

### Linux:

Make:

`make OS=linux`

## Run:

`./RTv1 scenes/default.rts`
